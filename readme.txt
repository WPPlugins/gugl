Capture search guery from Google and display button with additional text linked to destination page.


== Description ==

This plugin can capture a query string when the visitor comes to your Site from Google Results Page. Then you can show your visitor button with prepared text and search phrase to get high CTR.

= There is 8 default Color Schemes for button =
*	Grey
*	Green
*	Blue
*	Red
*	Orange
*	White
*	Purple
*	Pink

== Installation ==

1. Upload `gugl` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[gugl]` shortcode somewhere in your blog

== Frequently Asked Questions ==

= Which shortcode I have to use? =

`[gugl]`

= What attributes for shortcode can I use? =

* before - text shown before query phrase
* link - url to your destination page
* title - title of this anchor
* newtab - 'yes' if you want link to pop up in new tab
* color - One of color scheme

= What about (not provided)? =

Right now plugin doesn't handle with that.
